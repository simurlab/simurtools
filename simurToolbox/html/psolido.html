psolido

Syntax

psolido

Description

psolido draws a prism defined by its 8 corners.

Examples

% PSOLIDO: un prisma definido por sus 8 vértices.
P=psolido
P = 3×16
0 1.0000 1.0000 0.5500 0.5500 0.4500 0 0 0.4500 0.5500 1.0000 0 0.4500 0.4500 0.5500 1.0000 0 0 0 1.5000 1.5000 1.5000 0 0 1.5000 1.5000 0 0 1.5000 1.5000 1.5000 0 0 0 0.5000 0.5000 0 0 0 0.5000 0.5000 0.5000 0.5000 0.5000 0.5000 0 0 0
