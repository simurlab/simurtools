Robotics related tools

Simurtools Toolbox™ include tools to help in some specific problems related with robots. They are an addition to the Robotics Systems Toolbox of matlab.

Contents

  • Rigid Body 3D Representation
  • Robot Manipulators Models

Rigid Body 3D Representation

We have several tools to draw an object or a frame in 3D:

  • voxel: draw a rectangular box
  • psolido: define a prisma defined by corners
  • pframe: draw a frame xyz

Robot Manipulators Models

Tools to model and represent and robot arms, such as the ABB irb-120 in our laboratory:

  • kin_irb120: symbolic direct kinematics matrix for the robot irb-120
  • irb120vr: to use with the Matlab VR Toolbox
  • irb120simulink: model to use with Simulink
