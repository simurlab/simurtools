Modelo del robot irb120, representado en VR

Syntax

See example.

Description

To use with the Matlab VR Toolbox.

Examples

robotWorld = vrworld('robot_scene.wrl');
open(robotWorld);
rbt = importrobot('abb_irb_120d.urdf');
rbt.DataFormat = 'row';
n = vrinsertrobot(robotWorld,rbt);
vrdrawnow
view(robotWorld);
