Getting Started with SimurTools

Description

Esta toolbox está desarrollada para reutilizar y aprovechar el código creado en aplicaciones de biomecánica y procesamiento de señales.

System Requirements

Necesita que estén instaladas la toolbox de señales para que todas las funciones estén disponibles

Features

  • Funciones para la detección de máximos
  • Funciones para el cálculo de distancias del paso
  • Funciones par el cálculo de distancias mediante doble integración
  • Funciones para el cálculo de la orientación

Examples

  • Detección de máximos:
buscamaximos()
