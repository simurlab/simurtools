kin_irb120

Syntax

kin_irb(5)
kin_irb(6)

Description

kin_irb(5) gets the symbolic direct kinematics matrix for the robot irb-120, ignoring the last joint.
kin_irb(6) gets the symbolic direct kinematics matrix for the robot irb-120.
